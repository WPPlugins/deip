== Description ==

Replaces comment author ip by 127.0.0.1 after a configurable amount of days.

Storage of ip addresses prohibited by german privacy laws, which is the sole reason of this plugin.

== Installation ==

1. Upload the deip-folder to the `/wp-content/plugins/` directory or by upload in the webinterface.
2. Activate the plugin through the 'Plugins' menu in WordPress
